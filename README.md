## What Is OpenVPN, Actually?

OpenVPN has been around for over two decades, and at this point it's the protocol that most other VPN software is built on top of. The company behind it now sells two distinct products for businesses:

- **Access Server** — a self-hosted VPN you install on your own infrastructure (Linux, Docker, cloud VMs, etc.)
- **CloudConnexa** — a fully managed, cloud-delivered VPN service with Zero Trust networking baked in

They're genuinely different products for different types of teams. More on that in a moment.

The brand is trusted by some serious names: Google, Tesla, Amazon, Salesforce, Boeing, Cisco, Nvidia — the kind of customer list that at least tells you the thing works at scale.

Pricing is based on **simultaneous connections** (not users — connections). You can have unlimited users registered; only the ones actually connected at the same time count against your limit.

## CloudConnexa Pricing: Managed Cloud VPN with Zero Trust

CloudConnexa is the "we handle the infrastructure, you handle your team" option. It runs across 30+ worldwide Points of Presence, includes IDS/IPS, web filtering, DNS security, and Zero Trust access controls — all without you needing to manage a single server.

Pricing is per **seat** (users + connectors). A connector is a piece of software that makes your internal servers or networks accessible through CloudConnexa — so if you have 10 users and 2 connectors, you need 12 seats.

Annual plans save up to 20% versus monthly billing.

## Access Server vs. CloudConnexa: Which One Do You Actually Need?

This is where a lot of people get confused, so here's the honest version:

**Pick Access Server if:**
- You want to host everything yourself (air-gapped environments, strict data residency requirements)
- You need FIPS compliance, RADIUS authentication, or custom auth plugins
- You're comfortable managing Linux servers or cloud VMs
- You want the absolute lowest licensing cost and can handle the operational work yourself

**Pick CloudConnexa if:**
- You don't want to manage VPN servers at all
- You have teams in multiple locations that need to connect to each other or to cloud resources
- You want Zero Trust access controls, IDS/IPS, and web filtering without building that stack yourself
- You're scaling and want predictable costs without infrastructure overhead

The short version: Access Server gives you control. CloudConnexa gives you simplicity. Both give you the same underlying security model.
